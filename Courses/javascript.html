Quick Reference Notes

Variables & Data Types

var, let, const declarations. Strings, numbers, booleans, objects, arrays, null, undefined, and symbols.

Fundamentals

Operators

Arithmetic, comparison, logical, assignment, and ternary operators. Short-circuit evaluation and operator precedence.

Basics

Control Flow

if/else statements, switch cases, loops (for, while, do-while), break and continue statements.

Fundamentals

Functions

Function declarations, expressions, arrow functions. Parameters, return values, default arguments, and rest parameters.

Intermediate

Scope & Closures

Global and local scope, function scope, block scope. Closures, lexical environment, and data encapsulation.

Intermediate

Objects & Prototypes

Object literals, constructors, prototype chain. Methods, this binding, and prototypal inheritance.

Intermediate

Array Methods

map, filter, reduce, forEach, find, some, every, includes. Immutability patterns and method chaining.

Intermediate

DOM Manipulation

Selecting elements, creating/removing nodes. Modifying attributes, classes, styles, and innerHTML/textContent.

Intermediate

Event Handling

Event listeners, event objects, event bubbling/capturing. Delegation, preventDefault, and stopPropagation.

Intermediate

Promises & Async

Promise states, then/catch/finally. async/await syntax, error handling, and async patterns for API calls.

Advanced

Classes & OOP

Class syntax, constructors, methods, getters/setters. Inheritance, static methods, and encapsulation.

Advanced

Error Handling

try/catch/finally blocks, throwing errors, error types. Stack traces and debugging techniques.

Advanced

Intermediate

Advanced
